## 🔬 Methodology
Every finding is documented with:  
1. **Platform/Lab**  
2. **Context** (where the issue occurs)  
3. **Payload** (lab-only, safe)  
4. **Evidence** (screenshots)  
5. **Impact** (real-world risk)  
6. **Risk rating**  
7. **Root cause & Remediation**
